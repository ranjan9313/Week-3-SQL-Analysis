Subqueries Filtering with scalar & correlated subqueries
How it works
Scalar subquery (WHERE sales > (SELECT AVG(sales)…)) computes one value once and filters rows against it. Correlated subquery re-executes for every outer row, comparing against a per-customer MAX.
Above-Average Sales
Top Order / Customer
SELECT o.order_id, c.customer_name, p.category, ROUND(o.sales, 2) AS sales
FROM   orders o
JOIN   customers c USING(customer_id)
JOIN   products  p USING(product_id)
WHERE  o.sales > (SELECT AVG(sales) FROM orders)  -- scalar subquery
ORDER  BY o.sales DESC LIMIT 10;

CTEs Customer lifetime aggregation with WITH clauses
How it works
The WITH customer_sales AS (…) block pre-aggregates SUM/COUNT/AVG per customer. The outer query then joins dimensions — cleaner and reusable compared to inline subqueries.
WITH customer_sales AS (
    SELECT   customer_id,
             ROUND(SUM(sales),  2) AS total_sales,
             ROUND(SUM(profit), 2) AS total_profit,
             COUNT(DISTINCT order_id) AS num_orders
    FROM     orders
    GROUP BY customer_id
)
SELECT c.customer_name, c.segment, cs.total_sales, cs.num_orders, cs.total_profit
FROM   customer_sales cs
JOIN   customers c USING(customer_id)
ORDER  BY cs.total_sales DESC LIMIT 10;

Window Functions ROW_NUMBER and RANK for ranking and analysis
How it works
ROW_NUMBER() OVER (PARTITION BY customer_id ORDER BY sales DESC) assigns 1 to each customer's best order. RANK() assigns the same rank to ties, skipping the next rank — revealing true competitive position.
ROW_NUMBER — Best Order
RANK — Segment Ranking
WITH ranked AS (
    SELECT order_id, customer_id, order_date,
           ROUND(sales, 2) AS sales,
           ROW_NUMBER() OVER (
               PARTITION BY customer_id
               ORDER BY     sales DESC
           ) AS rn
    FROM   orders
)
SELECT c.customer_name, r.order_id, r.order_date, r.sales
FROM   ranked r JOIN customers c USING(customer_id)
WHERE  r.rn = 1
ORDER  BY r.sales DESC LIMIT 10;

Combined JOIN + CTE + Window Functions — full customer profile 
WITH sales_agg AS (
    SELECT   customer_id,
             ROUND(SUM(sales),  2) AS total_sales,
             ROUND(SUM(profit), 2) AS total_profit,
             COUNT(DISTINCT order_id) AS num_orders
    FROM     orders GROUP BY customer_id
),
ranked AS (
    SELECT *,
           RANK()   OVER (ORDER BY total_sales  DESC) AS sales_rank,
           RANK()   OVER (ORDER BY total_profit DESC) AS profit_rank,
           NTILE(4) OVER (ORDER BY total_sales  DESC) AS sales_quartile
    FROM   sales_agg
)
SELECT c.customer_name, r.total_sales, r.total_profit,
       r.sales_rank, r.profit_rank,
       CASE r.sales_quartile
           WHEN 1 THEN 'Top 25%'
           WHEN 4 THEN 'Bottom 25%' ...
       END AS customer_tier
FROM ranked r JOIN customers c USING(customer_id)
ORDER BY r.sales_rank LIMIT 10;
